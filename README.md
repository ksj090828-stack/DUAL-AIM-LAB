# DUAL AIM LAB 3D V11

3D 아레나를 고정 표적과 이동 표적 모드로 분리하고 배경을 정리한 버전입니다.

## 3D 아레나 모드
- 3D 고정 표적: 표적이 움직이지 않음
- 3D 이동 표적: 표적이 좌우·상하로 움직임
- 두 모드 모두 표적은 시간으로 자동 사라지지 않음
- 표적을 맞혔을 때만 새로운 표적으로 교체

## 배경 변경
- 네모난 벽 패널과 블록 장식 제거
- 부드러운 벽과 바닥 그라데이션 적용
- 원근선과 조명만 남긴 단순한 훈련 공간
- 기존보다 덜 복잡하고 표적이 잘 보이도록 조정

## 실행
압축을 푼 뒤 `index.html`을 Chrome 또는 Edge에서 실행하세요.


## V12 회피 및 원형 스킬샷 개선
- 회피 연습에 기존 직선 투사체 유지
- 플레이어 현재 위치에 생성되는 원형 장판 추가
- 화면을 가로지르는 ㅡ형 공격 추가
- 장판과 ㅡ형 공격은 경고 시간이 지난 뒤 발동
- 세 공격 패턴이 무작위로 섞여 등장
- 원형 스킬샷에 원형 진행 게이지와 남은 시간 표시
- 폭발 직전 테두리와 채우기 효과 강화
- 원형 스킬샷을 인식한 AI 회피 속도를 소폭 증가


## V12 감도 설정 추가
- 설정 메뉴에 마우스 DPI 입력 추가
- 게임 감도 입력 추가
- 기본값: 800 DPI / 감도 1.00
- 유효 감도는 `DPI ÷ 800 × 게임 감도`로 계산
- 2D 조준점 이동 속도에 적용
- 3D 아레나 카메라 회전 속도에 적용
- 설정값은 브라우저에 저장되어 다음 실행에도 유지
- 허용 범위: DPI 100~6400, 감도 0.05~5.00


## 마우스 화면 이탈 수정
- 게임 시작 시 브라우저 Pointer Lock 사용
- 실제 마우스 포인터가 게임 영역 밖으로 빠져나가지 않음
- ESC로 잠금 해제
- 게임 화면을 다시 클릭하면 잠금 재적용
- 일시정지, 결과 화면, 메뉴에서는 자동 해제
- 재개하면 다시 자동 고정
